===============================================
  Claude Code - Menu de Contexto do Explorer
===============================================

INSTALACAO:
  1. Dê duplo clique em "Instalar Claude Code.bat"
  2. Aceite a permissão de administrador
  3. Pronto!

USO:
  - Clique com botão direito em qualquer pasta no Explorer
  - Selecione "Claude Code"
  - Um formulário aparecerá com as opções:
      * Modo Administrador - abre o terminal como admin
      * Dangerously Skip Permissions - executa com --dangerously-skip-permissions
  - Clique "Iniciar" para abrir o terminal

DESINSTALACAO:
  1. Dê duplo clique em "Desinstalar Claude Code.bat"
  2. Aceite a permissão de administrador

REQUISITOS:
  - Windows 10 ou 11
  - Claude Code (CLI) instalado e disponível no PATH
  - PowerShell 5.1+ (já incluso no Windows)

ARQUIVOS:
  - Instalar Claude Code.bat    -> Instalador (duplo clique)
  - Desinstalar Claude Code.bat -> Desinstalador (duplo clique)
  - install.ps1                 -> Script de instalação
  - uninstall.ps1               -> Script de desinstalação
  - claude-code-launcher.ps1    -> Formulário GUI + lançador
  - claude-code.ico             -> Ícone (opcional)
